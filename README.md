# SniBlog
Source code of my blog. I upload random articles here about my experiences, daily life and everything I ever want to share üçÉ

### Tech Stack
- React.js
- Express.js
- MongoDB + Mongoose

##  Deployment Setup

#### Setup

Environment variables in `server/.env`
```
DB_USER= MongoDB user
DB_PASS= MongoDB password
PORT= Server Port
CLIENT_URL= Client Origin (eg. example.com)
```

Environment variables in `client/.env`
```/
VITE_SERVER= Backend endpoint (eg. api.example.com)
```
